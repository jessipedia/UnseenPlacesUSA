UnseenPlacesUSA API

This is the UnseenPlacesUSA API documentation. Real documentation will eventually go here. You can search the database in two ways:

Location

Location searches for places within a given area. You can search by state, district, and inhabited territories.

Example:

http://localhost:3000/api/place?location=Iowa

Search

Search searches the place descriptions. It returns all results that contain the search terms.

Example:

http://localhost:3000/api/place?search=gold
